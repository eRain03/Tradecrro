# Yahoo Finance 外汇行情工具

基于 Yahoo Finance API 的外汇行情获取工具，免费、无需 API Key。

## 功能列表

### 实时行情
- 获取单个/多个货币对实时价格
- 支持所有 Yahoo Finance 上的外汇对

### 历史数据
- 支持多种时间周期：1 分钟 ~ 1 年
- 数据包含：开盘价、最高价、最低价、收盘价、成交量

### 持续监控
- 定时刷新行情
- 自动保存数据到 JSON

## 快速开始

```bash
# 安装依赖
npm install axios

# 运行程序
node yahoo_forex_fetcher.js
```

## 菜单说明

| 选项 | 功能 |
|------|------|
| 1 | 获取默认 6 个货币对行情 |
| 2 | 自定义货币对行情 |
| 3 | 获取历史数据 |
| 4 | 持续监控模式 |
| 0 | 退出 |

## 货币对格式

使用 Yahoo Finance 格式，在货币对后加 `=X`：

```
EURUSD=X    - 欧元/美元
GBPUSD=X    - 英镑/美元
USDJPY=X    - 美元/日元
AUDUSD=X    - 澳元/美元
USDCAD=X    - 美元/加元
USDCHF=X    - 美元/瑞士法郎
NZDUSD=X    - 新西兰元/美元
```

## 历史数据参数

### 时间范围 (range)
- `1d`, `5d` - 1 天，5 天
- `1mo`, `3mo`, `6mo` - 1 月，3 月，6 月
- `1y`, `2y`, `5y`, `10y` - 1 年，2 年，5 年，10 年
- `ytd` - 年初至今

### 时间间隔 (interval)
- `1m`, `2m`, `5m`, `15m`, `30m` - 分钟级
- `60m`, `90m`, `1h` - 小时级
- `1d` - 日线
- `1wk` - 周线
- `1mo` - 月线

## 输出目录

所有数据保存到 `output/` 目录：
- `yahoo_quotes_*.json` - 实时行情
- `yahoo_history_*.json` - 历史数据
- `yahoo_monitor_*.json` - 监控数据

## 注意事项

1. **速率限制**：Yahoo Finance 有请求频率限制，建议：
   - 单次请求不要太频繁
   - 监控模式间隔 >= 60 秒

2. **数据延迟**：免费数据可能有 15-20 分钟延迟

3. **交易时间**：外汇市场 24 小时交易，但某些时段数据可能不更新

## 故障排除

### 429 Too Many Requests
- 等待 1-2 分钟再试
- 减少请求频率

### 无数据返回
- 检查货币对符号是否正确
- 某些小众货币对可能不支持
